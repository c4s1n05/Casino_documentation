Game Launch SOP & Checklist

End-to-end process for launching new games from a third-party aggregator into the live casino lobby. Covers the full workflow from aggregator request through to post-launch monitoring, with a per-game interactive checklist.

📅 May 2026
👤 Owner: Casino Manager
🔗 Aggregator: SoftSwiss
📋 Version: 1.0
ℹ️
Third-Party Aggregator

All games are sourced through SoftSwiss as the platform aggregator. The Casino Manager does not directly contract with individual game studios — all game requests, availability checks, metadata, assets and technical go-lives are managed through the SoftSwiss back office and account manager. Direct contact with individual studios should only occur for commercial discussions, and must be copied to your SoftSwiss account manager.

⚠️
Target Cadence

Minimum 4 new game launches per month. The Casino Manager should maintain a rolling 90-day launch calendar and keep at least 2 games in active pipeline with SoftSwiss at all times. Pipeline is reviewed weekly and reported monthly to the COO.

02
Working with the Aggregator (SoftSwiss)
🔴
Key Rule

All game technical issues — including games not loading, incorrect RTP display, bet limit errors, asset display problems or feed outages — must be raised as a ticket with SoftSwiss Support. Never attempt to directly contact the game studio for technical resolution. Always log the SoftSwiss ticket reference number in the incident record.

ScenarioActionSoftSwiss Contact
Game not showing in back officeConfirm activation request was submitted correctly. Raise support ticket with game name and request reference.SoftSwiss Support

Geo restriction issueRaise with account manager — SoftSwiss manages provider geo-block configuration. Do not go live until confirmed resolved.SoftSwiss AM
Game not loading for playersRaise urgent support ticket immediately. Monitor SoftSwiss status page. Post player-facing comms if outage exceeds 15 minutes.SoftSwiss Support (urgent)
RTP displaying incorrectlyDo not go live. Raise ticket with SoftSwiss immediately. If already live, set game to hidden until resolved.SoftSwiss Support (urgent)
