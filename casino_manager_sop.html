04
Lobby Management & Optimisation
Principle

The lobby is the primary commercial lever available to the Casino Manager. Every placement decision should be data-led and tested. No banner, category or feature position should remain static for more than 4 weeks without a performance review.

AreaStandard
Homepage Banner Maximum 5 featured game banners. Rotated fortnightly or when CTR drops below 25%. A/B test at least one creative variant at all times. Casino Manager approves all banner briefs.
Category Structure Minimum categories: New Games, Top Games, Slots, Live Casino, Table Games, Bonuses. Review category performance monthly and adjust based on player navigation data.
New Games Placement All new titles featured in 'New Games' category for a minimum of 30 days post-launch. High-priority launches also featured in homepage banner for first 2 weeks.
Game Retirement Any title generating less than 0.1% of GGR for 3 consecutive months to be reviewed. Casino Manager to assess whether to retire, reposition, or run a targeted promotion before removal.
Metadata & Search Game metadata (title, provider, category, tags) must be complete and accurate for all live titles. Casino Manager to audit metadata quarterly to support SEO and on-site search performance.
Mobile Optimisation All new titles must be verified for mobile performance via SoftSwiss QA tools before launch. Mobile GGR share tracked weekly — target greater than 60% of total GGR on mobile devices.

Escalation Matrix
SoftSwiss — Platform Escalation Route

SoftSwiss is the third-party platform provider. All game-related issues (outages, feed errors) and all payment-related issues must be routed through SoftSwiss support. The Casino Manager is responsible for raising the ticket, tracking resolution and escalating to the COO if SoftSwiss does not resolve within the agreed SLA window. Always retain the SoftSwiss ticket reference number in the incident log.
